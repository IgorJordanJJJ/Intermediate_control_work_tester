Чтобы создать две директории "Игрушки для школьников" и "Игрушки для дошкольников", откройте терминал и выполните следующие команды:

mkdir "Игрушки для школьников"
mkdir "Игрушки для дошкольников"
Чтобы создать текстовые файлы "Роботы", "Конструктор" и "Настольные игры" в каталоге "Игрушки для школьников", выполните следующие команды:

cd "Игрушки для школьников"
touch Роботы Конструктор "Настольные игры"
Чтобы создать текстовые файлы "Мягкие игрушки", "Куклы" и "Машины" в каталоге "Игрушки для дошкольников", выполните следующие команды:

cd ../"Игрушки для дошкольников"
touch "Мягкие игрушки" Куклы Тачки
Чтобы объединить эти два каталога в один "Имя игрушки", выполните следующую команду:

cd ...
mv "Игрушки для школьников" "Имя игрушки"
mv "Игрушки для дошкольников" "Имя игрушки"
Чтобы переименовать каталог "Имя игрушки" в "Игрушки", выполните следующую команду:

mv "Имя игрушки" Игрушки
Чтобы просмотреть содержимое каталога "Toys", выполните следующую команду:

ls Toys
Чтобы установить и удалить пакет snap, вы можете использовать следующие команды. В этом примере мы установим пакет "gnome-calculator", а затем удалим его:

sudo snap install gnome-calculator
sudo snap remove gnome-calculator
Чтобы добавить случайное задание, выполняемое каждые 3 минуты, вы можете создать простой сценарий bash, который записывает что-то в текстовый файл или копирует файл из каталога A в каталог B, а затем использовать планировщик cron для запуска сценария каждые 3 минуты. Вот пример сценария, который записывает текущую дату и время в текстовый файл:

#!/bin/bash
echo "$(дата)" >> ~/mylogfile.txt
Сохраните этот скрипт под именем mytask.sh в домашнем каталоге (~/). Затем откройте таблицу cron, выполнив следующую команду:


crontab -e
Добавьте следующую строку в нижнюю часть файла, чтобы запускать скрипт каждые 3 минуты:


*/3 * * * * ~/mytask.sh
Сохраните файл и выйдите из редактора. Теперь каждые 3 минуты будет выполняться сценарий mytask.sh, а текущие дата и время будут добавлены в файл mylogfile.txt в вашем домашнем каталоге.
